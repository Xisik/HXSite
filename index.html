Core Engine Features

Responsive Canvas Renderer

Powered by a custom QWidget canvas, ensuring smooth canvas rotation, zooming, and rendering even with large documents and complex layers.

Dynamic Brush Engine

The BrushEngine natively supports Wintab, X11, and Wayland tablet inputs via Qt's TabletInputEvent, providing zero-latency pressure sensitivity and tilt response.

Layer Management

A robust CanvasDocument and LayerManager system allows for infinite non-destructive layers with real-time composite blending, opacity, and locking controls.

Non-destructive Workflow

Make mistakes fearlessly. The CommandManager implements an exact Command Pattern, taking lightweight memory snapshots for infinite Undo and Redo operations.

Native .hxp Format

Save your complete workspace using the efficient, JSON-based .hxp project format handled by ProjectSerializer, or export directly to PNG/JPG.

Modular Architecture

Built with C++20 standard, the engine strictly separates UI, Input Mapping, and Rendering layers, making it highly stable and immune to UI-thread blocking.

System Requirements

Windows

  • OS: Windows 10 or Windows 11 (64-bit)
  • Processor: x86_64 architecture CPU
  • Graphics: Standard Windows graphics stack
  • Memory: 4GB RAM minimum (8GB+ recommended)
  • Input: Mouse or Wintab/Windows Ink compatible drawing tablet
  • Dependencies: None! All required Qt6 and MinGW DLLs are bundled.

Linux

  • OS: 64-bit Modern Distribution (Ubuntu 22.04+, Fedora 38+, Arch)
  • Display Server: Wayland or X11
  • Graphics: Standard Qt-supported display stack
  • Memory: 4GB RAM minimum (8GB+ recommended)
  • Input: Mouse or libinput compatible drawing tablet
  • Dependencies: glibc 2.31+. Required Qt6 .so libraries are bundled locally.
