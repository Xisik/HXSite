About HXPainter

HXPainter is a passion project born from the desire for a lightweight, high-performance painting tool that respects the user's freedom. Built with C++ and the Qt framework, it leverages modern GPU acceleration to provide a seamless creative experience.

The application is organized around a real painting workflow: a document model for layered canvases, brush and eraser engines, project serialization, PNG/JPG export, tablet input mapping, and command-based history for reliable Undo and Redo.
