Get HXPainter Today

Ready to start creating? Download the latest standalone HXPainter archive for your platform.

The Windows and Linux packages are compressed .tar.xz archives. Extract the archive, then run HXPainter.exe on Windows or HXPainter on Linux.

Download HXPainter for Windows Download HXPainter for Linux

System Requirements:

  • Windows: Windows 10 or later (64-bit)
  • Linux: 64-bit desktop distribution with Wayland or X11
  • Graphics: Standard Qt-supported display stack
  • Processor: Intel Core i3 or equivalent
  • Memory: 4 GB RAM
  • Storage: 150 MB available space

Source Code and License:

The HXPainter source is available on GitHub under GPL-3.0, with C++/Qt code, CMake build files, scripts, resources, and third-party notices.
